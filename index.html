Pages

Each JS file in the /pages folder is a page.

  • pages/index.js
  • pages/about.js
  • pages/account/settings.js

Pre-rendering modes

  • Static generation
  • Server-side rendering

Static generation

HTML is generated at build time

Good for websites with static data, or with data that doesn't change often.

Server-side rendering

HTML is generated on each request

Good for websites with data that changes often, or web apps with authentication.

Data fetching

  • getStaticProps
  • getServerSideProps

Dynamic routes

  • pages/users/[id].js
  • pages/[username]/posts.js

Whatever is in [] will be passed to the data-fetching functions.

For static generation, getStaticPaths is also necessary.
